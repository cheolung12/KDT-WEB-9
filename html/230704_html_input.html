input 태그

사용자에게 값을 입력받을 때 사용, inline 요소

  • name 속성 : 무엇을 받을것인지 명시, 백엔드에 정보 전송할때 사용 ex) phone, address ...
  • placeholder 속성 : 사용자가 무슨 값을 입력해야 하는지 알려주는 일종의 안내메시지 역할, 연한 글씨체로 표시
  • value 속성 : input태그의 값을 지정, text에서는 기본값으로 button에서는 기능 명시용
